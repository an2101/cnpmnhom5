MY GROUP PROJECT

C++-Based Online Shopping Platform

Functionality:
This project involved the development of an online shopping platform using C++. The platform provides users with a seamless shopping experience, allowing them to browse products, add items to their cart, and complete purchases. The system includes both a backend to manage data and a frontend for user interaction.
Outcomes:
The platform was successfully deployed, offering a stable and user-friendly shopping experience. It efficiently handles multiple users, ensuring smooth transactions and data management. The project highlighted the successful integration of full-stack development, with strong backend support and an intuitive user interface.

H-Bridge Motor Control Circuit

Functionality:
The H-Bridge Motor Control Circuit was designed to control the direction and speed of a DC motor. This system allows for precise control of the motor, enabling it to operate in both forward and reverse directions.
Outcomes:
The circuit was successfully developed and tested, demonstrating reliable and accurate motor control under various conditions. The project showcased the ability to integrate hardware design with microcontroller programming, resulting in a smooth and efficient motor control system.
